О проекте

Это демонстрация проекта для бесплатного доступа к функциям ChatGPT без необходимости платных API ключей. Проект использует библиотеку G4F для доступа к различным моделям AI.

Генератор изображений использует Unsplash API для создания изображений по текстовому описанию, а затем процессинг с помощью библиотеки Sharp и трассировку в SVG формат с помощью Potrace.
